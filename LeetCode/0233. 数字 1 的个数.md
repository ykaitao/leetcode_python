# [233. 数字 1 的个数](https://leetcode.cn/problems/number-of-digit-one/)

给定一个整数 n，计算所有小于等于 n 的非负整数中数字 1 出现的个数。

 

示例 1：
```
输入：n = 13
输出：6
```
示例 2：
```
输入：n = 0
输出：0
```

提示：
```
0 <= n <= 10**9
```

代码：
```python3
class Solution:
    def countDigitOne(self, n: int) -> int:
        s = str(n)
        N = len(s)
        hi, lo, cur = 0, n, 0
        digit = 10**(N - 1)
        res = 0
        for i in range(N):
            cur = int(s[i])
            lo = lo % digit
            if cur == 0:
                res += hi * lo
            elif cur == 1:
                res += lo + 1 + hi * digit
            else:
                res += (hi + 1) * digit
            
            digit //= 10
            hi = hi * 10 + cur
        return res
```
